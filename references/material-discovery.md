# Material Discovery

当工作区里除了内置课程资料之外，还混有很多老师文件、照片、脚本或旧草稿时，用这份清单定位真正有用的材料。

## 1. 先用仓库内置资料打底

优先从仓库自身读取：

- `course-materials/绪论课ppt.pdf`
- 当前实验对应的主题实验 PDF
- 相关课程基础知识 PDF

只有这些内置资料无法覆盖老师补充要求时，才继续扩大到工作区其他文件。

## 2. 找老师补充材料

在工作区搜索与当前实验同名或近似命名的文件，重点关注：

- `pdf`
- `ppt` / `pptx`
- `docx`
- `md`

老师补充材料通常可能包含：

- 更新后的实验步骤
- 特定仪器常数
- 简化版记录表
- 当次课特别强调的作图或讨论要求

如果老师补充材料和主题实验 PDF 有差异，优先采用更晚、更具体、与本次实验直接相关的老师材料。

## 3. 找原始数据

搜索：

- `jpg` / `jpeg` / `png` / `heic`
- 扫描件
- Excel / CSV
- 手写表格照片
- 实验记录本照片

如果照片很多，先按时间戳或页码排序，再开始识别。

## 4. 复用本地模板或脚本时要先核对

本地已有的 `tex`、`py`、`csv` 文件只有在满足以下条件时才可复用：

- 属于同一个实验
- 公式与本次指导书一致
- 不是旧版、不是从别的实验复制来的

不要因为某个文件已经存在，就默认它是正确答案。
